## 📋簡單介紹

這裡存放著我在[ZeroJudge](https://zerojudge.tw)的所有解題記錄,每一題會開一個資料夾來存放各種語言的解法(C / C++ / Java / Python).目前大部分題目都只有 C++ 的解法,如果條件允許的話後續會逐漸補上其他語言的解法.針對解題過程的部分我都會盡可能去撰寫解題報告,可能會有以下內容:
- 會用到的演算法
- 會用到的資料結構
- 需要注意的問題(測資問題,輸入輸出要注意的點...)
- 在討論區收集到的刁鑽測資
